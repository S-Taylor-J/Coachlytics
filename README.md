# 📊 Coachlytics

**Coachlytics** is an iOS app designed to help coaches track players, monitor performance, and manage team insights all in one place.

Built with **Swift and Xcode**, the app focuses on making coaching data easier to organize and analyze directly from your phone.

---

## 🚀 Features

* 👥 **Player Management** – Keep track of players and team members
* 📈 **Performance Tracking** – Monitor player progress and statistics
* 📝 **Coaching Notes** – Record observations and feedback
* 📱 **Native iOS Interface** – Clean, responsive design
* 💾 **Local Data Storage** – Data stored securely on-device

---

## 🛠 Tech Stack

| Technology          | Purpose                 |
| ------------------- | ----------------------- |
| **Swift**           | App development         |
| **Xcode**           | Development environment |
| **UIKit / SwiftUI** | User interface          |
| **Local Storage**   | Persistent data storage |

## ⚙️ Getting Started

### Clone the repository

```
git clone https://github.com/S-Taylor-J/Coachlytics.git
```

### Open the project

Open in **Xcode**:

```
Coachlytics.xcodeproj
```

### Run the app

1. Select an **iOS Simulator or device**
2. Press **Run ▶ in Xcode**
